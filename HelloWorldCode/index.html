HTML HyperText Markup Language
É a principal linguagem de quase todo o conteúdo da web. A maioria do que você vê na tela de seu navegador está descrita, fundamentalmente, usando HTML. Mais precisamente, o HTML é a linguagem que descreve a estrutura e o conteúdo semântico de um documento da Web.
HTML é um padrão internacional cuja especificações são mantidas pelo World Wide Web Consortium.
